EVENIMENT
Asociaţia Studenţilor Informaticieni Ieşeni (ASII) este reprezentata de un grup de persoane determinate, creative si dinamice, ce au in comun pasiunea pentru Informatica si impartasesc un set de teluri si valori. Activitatile organizate in cadrul asociatiei se adreseaza studentilor si profesorilor Facultatii de Informatica Iasi, dar si comunitatii IT din Romania, dorind sa promoveze proactivitatea, dezvoltarea personala si sociala.
